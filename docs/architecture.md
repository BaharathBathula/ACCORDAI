# ACCORDAI Architecture (v0.1)

## Goal
Build a modern cloud-native Agency Management System (AMS) for independent insurance agencies.

## MVP Modules
- Agency Core (Clients, Policies)
- Documents
- Workflows (Tasks, Renewals)
- Quoting (Phase 1)
- Reporting

## Tech Stack (initial)
- Backend: FastAPI (Python)
- DB: PostgreSQL (Phase 2)
- Frontend: Next.js (Phase 2)
- Cloud: AWS (target)
